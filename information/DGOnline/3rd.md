# 📃 DGOnline 介绍

DGOWorld 3rd 是 DGO 的外服，开放于`2021年7月6日`。

**服务器版本** <br/>
`基岩版 1.17.40.04`

**世界设置** <br/>
死亡掉落 / 生物破坏 / 火焰蔓延 / TNT 爆炸 / 开启坐标
